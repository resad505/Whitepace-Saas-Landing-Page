# Whitepace SaaS Landing Page

This is a practice project where I built a landing page for a fictional SaaS product called **Whitepace**. The primary goal of this project was to practice and improve my skills in structuring web pages and styling them using modern CSS techniques, with a strong focus on **Flexbox**.

## 🚀 Features

- Structured and semantic HTML5 markup.
- Responsive and flexible layouts built primarily with CSS Flexbox.
- Custom Google Fonts integration (Inter font family).
- Distinct sections including a Hero area, Project Management, Work Together, and an Extension section.
- Implementation of distinct visual elements like highlighted text backgrounds and background images.

## 🛠️ Technologies Used

- **HTML5**: For semantic structuring of the content.
- **CSS3**: For styling, layout design, and achieving the visual aesthetic.
- **Flexbox**: Extensively used for aligning and distributing space among items in the header, navigation, and main content sections.
- **Ionicons**: For integrating vector icons (e.g., arrows in buttons).

## 📂 Project Structure

- `index.html`: The main HTML document containing the structure of the landing page.
- `styles.css`: The stylesheet containing all the CSS rules, including Flexbox layouts and typography definitions.
- `assets/`: A directory containing all the images and logos used in the project.

## 💻 How to Run Locally

1. Clone or download the repository to your local machine.
2. Navigate to the project folder.
3. Open the `index.html` file in any modern web browser to view the project. No build tools or servers are required!

## 🔗 Live Site

You can view the live deployment of this project here: [Whitepace SaaS Landing Page](https://whitepace-saas-landing-page.vercel.app/)

## 🎯 Learning Objectives

This project was built to practice:
- Creating complex layouts using `display: flex`.
- Aligning items using `justify-content` and `align-items`.
- Using absolute and relative positioning for decorative elements (like text highlights).
- Structuring CSS using logical classes and basic BEM principles.
